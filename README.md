  #🧁 Cupcake Ordering App

This is an Android application developed as part of my internship, demonstrating the implementation of a complete cupcake ordering workflow. The app provides a simple, user-friendly interface for selecting cupcake options and placing an order.

📱 Features
Select quantity of cupcakes
Choose from multiple flavors
Pick a convenient pickup date
View order summary with real-time subtotal calculation
Smooth navigation across multiple screens
🛠️ Tech Stack
Language: Kotlin
Platform: Android
UI Design: Material Design principles
Architecture Concepts: State management, Navigation
🎯 Learning Outcomes
Managing UI state across multiple screens
Implementing navigation between screens
Handling and validating user input
Designing clean and intuitive user interfaces
Building a complete end-to-end app workflow
